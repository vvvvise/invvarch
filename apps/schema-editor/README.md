# @vvvvise/invvarch-schema-editor

## Usage
```sh
$ yarn prisma:init
$ yarn prisma:generate
$ yarn prisma:migrate
$ yarn test

# kick from WorkspacesRoot
$ cd ../..
$ yarn schema-client prisma:init
$ yarn schema-client prisma:generate
$ yarn schema-client prisma:migrate
$ yarn schema-client test
```
